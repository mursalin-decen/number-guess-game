# 🎮 Number Quest: Ultra Neon V2

**Number Quest: Ultra Neon** is a high-end, 2-player competitive number guessing game built with Python. By blending classic gameplay with modern "Synthwave" aesthetics and smooth animations, it delivers a premium gaming experience directly in your terminal/IDE.

---

## ✨ Features

- **Neon Aesthetics:** High-contrast glowing borders and a "Glassmorphism" UI design.
- **Custom Player Profiles:** Enter personalized names for both players before the match starts.
- **Advanced Animations:** 
  - **Floating UI:** Cards hover smoothly to create a sense of depth.
  - **Pulsing Neon Waves:** Real-time wave physics at the bottom of the screen.
  - **Dynamic Starfield:** An animated background that adds life to the dark theme.
- **Competitive Scoring:** Tracks attempts for both players and declares a champion based on efficiency.
- **Full Reset Logic:** Pressing `R` allows a complete fresh start, including name re-entry.
